AeroAid - Emergency Drone Response

×

About This Project

AeroAid is an innovative emergency response application that connects drone operators with emergency situations in real-time. Built with React and modern web technologies, it provides a comprehensive platform for coordinating emergency assistance through drone technology.

Key Features

  • Real-time geolocation tracking and mapping
  • Efficient emergency assignment algorithms
  • Mobile-first responsive design for field use
  • Firebase Authentication and user management
  • Interactive map with OpenStreetMap integration
  • Real-time notifications and alerts via Firebase Cloud Messaging
  • Emergency status tracking and updates
  • Complex state management using React Context API

Technology Stack

React Context API Tailwind CSS Leaflet.js OpenStreetMap Firebase Firestore Geolocation API

Challenges & Solutions

Challenge: Implementing real-time location tracking for emergency scenarios while maintaining accuracy and performance.

Solution: Utilized the HTML5 Geolocation API combined with Leaflet.js for accurate mapping and real-time position updates, with Firebase Firestore for real-time database synchronization.

Challenge: Creating an efficient assignment algorithm for matching emergencies with available drone operators.

Solution: Developed a proximity-based algorithm considering operator availability, distance, and emergency priority levels using efficient search algorithms.
